\# টাইপস্ক্রিপ্টের Types এবং Interface এর মধ্যে পার্থক্য 



আমরা যখন টাইপস্ক্রিপ্ট দিয়ে প্রোগ্রামিং করি তখন প্রায়ই আমাদের Type এবং Interface নিয়ে কাজ করতে হয়। মূলত দুইটার উদ্দেশ্যই ডেটা কে কাঙ্খিত শেইপ দেওয়া কিন্তু এই দুইটি ডেক্লারেশন এর মধ্যে বেশ কিছু পার্থক্য আছে যা আমাদের কোডের আর্কিটেকচার, রিডেবিলিটি এবং স্কেলেবিলিটির উপর প্রভাব ফেলে





\## Type এবং Interface এর উদ্দেশ্য



\*\*Type\*\*: এটি একটি সাধারন টাইপ ডেক্লারেশন টুল যা প্রিমিটিভ টাইপের ডাটাগুলোকে ডিক্লেয়ার করতে ব্যবহার হয় এবং সেই ডেটা টাইপ গুলোর মধ্যে Union, Intersection প্রয়োগ করতে ব্যবহার করা হয়।

\*\*Interface\*\*: Interface কে মূলত object  এর গঠন নির্ধারণ করতে ব্যবহার করা হয়। অন্যভাবে বলা যায় Interface হলো object এর জন্য ব্লুপ্রিন্ট।  



```

type Id = number | string;



interface User {

&nbsp; name: string;

&nbsp; age: number;

&nbsp; id: Id

}

```
